# إضافة اللوجو

## الخطوات:

1. ضع صورة اللوجو في مجلد `public/` باسم:
   - `logo.png` (مستحسن)
   - أو `logo.jpg`
   - أو `logo.webp`

2. ضع صورة اللوجو الأبيض للخلفية باسم:
   - `logo-white.png`
   - أو `logo-white.jpg`

3. بعد إضافة الصور، شغّل:
   ```bash
   npm run build:client
   ```

## أسماء الملفات المطلوبة:

- `public/logo.png` - اللوجو الرئيسي (M وردي)
- `public/logo-white.png` - اللوجو الأبيض للخلفية الشفافة
