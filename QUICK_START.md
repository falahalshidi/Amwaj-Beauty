# دليل البدء السريع - Amwaj Beauty

## خطوات التشغيل

### 1. تثبيت الحزم
```bash
npm install
```

### 2. تشغيل المشروع
```bash
npm run dev
```

سيتم فتح:
- Frontend: http://localhost:3000
- Backend: http://localhost:5000

## استخدام الموقع

### 1. إنشاء حساب
- اذهب إلى صفحة "إنشاء حساب"
- أول حساب يتم إنشاؤه يصبح تلقائياً **مدير (Admin)**
- يمكن للمدير الوصول إلى لوحة التحكم

### 2. إضافة منتجات (للمدير فقط)
- سجّل دخول بحساب المدير
- اذهب إلى لوحة التحكم
- اضغط "إضافة منتج جديد"
- املأ البيانات:
  - اسم المنتج
  - الوصف
  - السعر (بالريال العماني)
  - الكمية
  - رابط الصورة (اختياري)

### 3. الطلب
- تصفح المنتجات
- اضغط "اطلب الآن" على أي منتج
- املأ معلومات الشحن
- تأكيد الطلب

### 4. إدارة الطلبات (للمدير)
- في لوحة التحكم، اذهب لتبويب "إدارة الطلبات"
- يمكنك تغيير حالة الطلب:
  - قيد الانتظار
  - تم الإعداد
  - تم الشحن
  - مكتمل

## ملاحظات مهمة

- البيانات محفوظة في `data/` كملفات JSON
- عند إضافة طلب، الكمية تنقص تلقائياً من المخزون
- يمكنك استبدال صورة اللوجو في `public/logo.svg`

## استكشاف الأخطاء

### المشكلة: Backend لا يعمل
- تأكد من تثبيت الحزم
- تأكد من أن البورت 5000 غير مستخدم

### المشكلة: Frontend لا يعمل
- تأكد من تثبيت الحزم
- تأكد من أن البورت 3000 غير مستخدم

### المشكلة: لا يمكن تسجيل الدخول
- تأكد من أن Backend يعمل
- تحقق من console للأخطاء
