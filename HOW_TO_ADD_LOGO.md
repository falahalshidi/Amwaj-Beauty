# كيفية إضافة صور اللوجو

## الخطوات السريعة:

1. **ضع صورة اللوجو الرئيسية** في مجلد `public/`:
   - اسم الملف: `logo.png` (مستحسن)
   - أو `logo.jpg`
   - الصورة: حرف M وردي (من الصور التي أرسلتها)

2. **ضع صورة اللوجو الأبيض** للخلفية في مجلد `public/`:
   - اسم الملف: `logo-white.png`
   - أو `logo-white.jpg`
   - الصورة: نفس اللوجو لكن أبيض أو شفاف

3. **بعد إضافة الصور:**
   ```bash
   npm run build:client
   ```

## الملفات المطلوبة:

```
public/
  ├── logo.png          ← اللوجو الرئيسي (M وردي)
  └── logo-white.png    ← اللوجو الأبيض للخلفية
```

## ملاحظات:

- إذا لم تضع الصور، سيستخدم الكود SVG كـ fallback
- الصور يجب أن تكون بصيغة PNG أو JPG
- حجم الصورة: 200x200 بكسل أو أكبر (سيتم تصغيرها تلقائياً)

## بعد إضافة الصور:

الكود جاهز! فقط:
1. ضع الصور في `public/`
2. شغّل `npm run build:client`
3. الصور ستظهر تلقائياً في جميع الصفحات
